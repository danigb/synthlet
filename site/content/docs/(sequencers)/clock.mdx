`@synthlet/clock`

Generates a clock signal at a given tempo.

```ts
import { registerClockWorklet, createClockNode } from "synthlet";

const audioContext = new AudioContext();
await registerClockWorklet(audioContext);
const clock = createClockNode(audioContext, { bpm: 10 });
```

## Example

Trigger a kick with a clock:

```ts
import { registerSynthlet, KickDrum } from "synthlet";

const audioContext = new AudioContext();
await registerSynthlet(audioContext);
const clock = createClockNode(audioContext, { bpm: 120 });
const kick = createKickDrum(audioContext, { trigger: clock });
kick.connect(audioContext.destination);
```

## Parameters

- `bpm`: beats per minute
