`@synthlet/clock`

Generates a clock signal at a given tempo.

```ts
import { registerClockWorkletOnce, createClockNode } from "synthlet";

const audioContext = new AudioContext();
await registerClockWorkletOnce(audioContext);
const clock = createClockNode(audioContext, { bpm: 10 });
```

## Example

Trigger a kick with a clock:

```ts
import { registerSynthletOnce, KickDrum } from "synthlet";

const audioContext = new AudioContext();
await registerSynthletOnce(audioContext);
const clock = createClockNode(audioContext, { bpm: 120 });
const kick = KickDrum(audioContext, { trigger: clock });
kick.connect(audioContext.destination);
```

## Parameters

- `bpm`: beats per minute
