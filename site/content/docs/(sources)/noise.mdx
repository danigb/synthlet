A Noise generator. Currently only white noise is supported.

Example:

```ts
import { registerNoiseWorkletOnce, createNoiseNode } from "synthlet";

const audioContext = new AudioContext();
await registerNoiseWorkletOnce(audioContext);

const noise = createNoiseNode(audioContext, {
  type: "white",
});
noise.connect(audioContext.destination);
```
