`@synthlet/impulse`

This module generates an impulse when a gate is opened. Useful to generate a click sound.

```ts
import { registerImpulseWorkletOnce, createImpulseNode } from "synthlet";

const audioContext = new AudioContext();
await registerImpulseWorkletOnce(audioContext);
const impulse = createImpulseNode(audioContext);

impulse.gate.value = 1;
```

## Parameters

- `gate`: The gate trigger. 0 means off, 1 means on.
